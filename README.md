# PHP version

Проект распакован из `Тестовое задание.rar` и переведен в PHP-структуру. Публичные страницы также приведены к нормальной UTF-8 кодировке.

## Публичная верстка

Основные PHP-страницы находятся в:

`Верстка/template`

- `index.php`
- `categ.php`
- `view.php`
- `contact.php`

CSS, JavaScript, шрифты и изображения остались в исходных папках `css`, `js`, `fonts`, `img`.

## Админка

Рабочая страница админ-панели находится в:

`Админка/1/admin_1_rounded/site_content.php`

Она управляет настройками сайта, меню, боковым меню, новостями, календарем событий, галереей, клиентами, внешними ссылками, соцсетями, футером, формами и опросом. Данные сохраняются в:

`Верстка/template/data/content.json`

Публичные страницы читают данные из этого файла.

Заявки с форм сайта сохраняются в:

`Верстка/template/data/submissions.json`

## Редактор

CKEditor распакован в:

`Редактор/ckeditor`

Эта часть является готовой JS-библиотекой и не требует перевода в PHP.

## Запуск

При наличии PHP можно запустить публичную часть так:

```powershell
cd "Верстка/template"
php -S localhost:8000
```

После этого открыть `http://localhost:8000/index.php`.

Админку можно запустить отдельно:

```powershell
cd "Админка/1"
php -S localhost:8001
```

После этого открыть `http://localhost:8001/index.php`.
